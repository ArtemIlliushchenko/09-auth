# 📝 NoteHub

**NoteHub** is a simple and efficient web application for creating, storing, and managing personal notes.  
It helps keep your thoughts organized and accessible whether you are at home or on the go.  

---

## 🌟 Features

- Create, edit, and delete notes easily  
- Categorize notes by type (Todo, Personal, Work, Shopping, Meeting)  
- Browse notes with a clean and intuitive interface  
- Keyword search for quick access to notes  
- User authentication and profile management
